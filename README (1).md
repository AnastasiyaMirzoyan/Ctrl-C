# Код факультета

Интерактивная веб-игра для абитуриентов: 7 направлений подготовки, мини-игры,
сбор «модулей», статистика прохождения и рекомендация направлений.

## Структура проекта

```
kod-fakulteta/
├── index.html      # точка входа (подключает style.css и game.js)
├── style.css       # всё оформление
├── game.js         # вся логика игры
└── images/         # картинки (портреты, робот, фоны)
    ├── aleksey.png
    ├── alina.png
    ├── robot.png
    ├── bg-lab.jpg
    ├── bg-lobby.jpg
    └── bg-finale.jpg
```

## Как запустить локально

Просто открой `index.html` в браузере (двойной клик). Прогресс
сохраняется в браузере через `localStorage`.

## Как выложить на GitHub + GitHub Pages

1. Создай новый репозиторий на GitHub (например, `kod-fakulteta`).
2. Загрузи в него **всё содержимое этой папки** (файлы `index.html`,
   `style.css`, `game.js` и папку `images/`) — структуру не меняй.
3. В репозитории открой **Settings → Pages**.
4. В разделе *Build and deployment* выбери **Source: Deploy from a branch**,
   ветку `main` и папку `/ (root)`, нажми **Save**.
5. Через минуту игра будет доступна по адресу
   `https://<твой-логин>.github.io/<имя-репозитория>/`.

> Важно: имена файлов и папки `images/` должны остаться прежними —
> на них ссылается `game.js`.
