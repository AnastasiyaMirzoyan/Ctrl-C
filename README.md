# Код факультета

Интерактивная веб-игра для абитуриентов: 7 направлений подготовки, мини-игры,
сбор «модулей», статистика прохождения и рекомендация направлений.

## Структура проекта

```
kod-fakulteta/
├── index.html      # точка входа (подключает style.css и game.js)
├── style.css       # всё оформление
├── game.js         # вся логика игры
└── images/         # картинки (портреты, робот, фоны)
    ├── aleksey.png
    ├── alina.png
    ├── robot.png
    ├── bg-lab.jpg
    ├── bg-lobby.jpg
    └── bg-finale.jpg
```
